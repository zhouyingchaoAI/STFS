# 侯家塘地铁站客流分析脚本使用说明

## 功能
绘制近三年"侯家塘"地铁站客流情况，包含：
- 日客流趋势图
- 月度平均客流对比
- 年度客流对比
- 月度分布热力图

## 使用方法

```bash
cd /STFS_V1
python script/dd.py
```

## 输出
- 图表文件：`侯家塘_近3年客流分析.png`（保存在脚本运行目录）
- 控制台输出：详细的统计摘要信息

## 依赖包
- pandas
- pymssql
- matplotlib
- numpy

如果缺少依赖，请安装：
```bash
pip install pandas pymssql matplotlib numpy
```

## 常见问题

### 1. 数据库连接失败
- 检查数据库服务器是否可访问（10.1.6.230:1433）
- 检查网络连接
- 确认数据库配置是否正确

### 2. 图表不显示
- 脚本会自动保存图表到PNG文件
- 在无GUI环境中运行时，图表不会弹出窗口，但会保存到文件

### 3. 中文显示乱码
- 脚本已自动配置中文字体
- 如果仍有问题，请检查系统是否安装了中文字体

## 修改站点名称
如需分析其他站点，修改脚本中的 `STATION_NAME` 变量：
```python
STATION_NAME = "站点名称"
```
